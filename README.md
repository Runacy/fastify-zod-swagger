# fastify-zod-swagger

This is sample API that outputs OpenAPI specifications based on the validation schema defined by [Zod]().

Check the example of Swagger-UI from here.  
https://joe-king-sh.github.io/fastify-zod-swagger/


## Getting Started

### Prerequisites

```bash
$ node --version
node v18.2.1
```

### Installing

```bash
$ npm i
```

### Run

```bash
$ npm run dev
```
